Data type in java
    ข้อมูลเป็นส่วนสำคัญของโปรแกรม เพราะเราจะประมวลผลตามข้อมูลที่เรามี
    โดยเมื่อโปรแกรมทำงาน มันจะเก็บข้อมูลไว้ในหน่วยความจำ (ram)
    โดยข้อมูลที่จัดเก็บอยู่ในหน่วยความจำ เราสามารถเข้าถึงได้ผ่านสิ่งที่เรียกว่า ตัวแปร(variable)

    ทำให้ในทุกครั้งที่เราจะจัดเก็บข้อมูล เราจะต้องประกาศตัวแปรขึ้นมาก่อน
    และจะต้องมีการระบุชนิดของข้อมูลที่เราต้องการจะเก็บด้วย

    Primitive Data Type 
        เป็นชนิดของข้อมูลดั้งเดิมที่มีอยู่ในภาษา java โดยจะประดอบด้วย 4 หมวดหมู่
            1. Integer
                    เป็นชนิดข้อมูลสำหรับจัดเก็บเลขจำนวนเต็ม โดยประกอบด้วย 
                        1.1 byte มีขนาด 1 byte
                        1.2 short มีขนาด 2 byte
                        1.3 int มีขนาด 4 byte
                        1.4 long มีขนาด 8 byte
            2. Floating Point
                    เป็นชนิดข้อมูลสำหรับจัดเก็บเลขจำนวนเต็มที่มีทศนิยม โดยประกอบด้วย
                        2.1 float มีขนาด 4 byte 
                        2.2 double มีขนาด 8 byte
            3. Character
                    เป็นชนิดข้อมูลสำหรับจัดเก็บตัวอักษร ใช้ชนิดข้อมูลเป็น char และมีขนาด 2 byte
            4. Boolean
                    เป็นชนิดข้อมูลสำหรับจัดเก้บค่าจริงเท็จ ใช้ชนิดข้อมูลเป็น boolean 
                    โดยขนาดของข้อมูลจะขึ้นอยู่กับ environment ที่ jvm กำลังทำงานได้
                    แต่มีขั้นตำคือ 1 
    
    ทุก ๆ ชนิดข้อมูล จะมี build-in class สำหรับจัดการกับข้อมูลชนิดนั้น ๆ
    เช่น int จะมี build-in class คือ Integer ซึ่งจะมาเพิ่มความสามารถให้กับ int อีกที
