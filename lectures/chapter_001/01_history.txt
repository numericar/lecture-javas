
    java มี 2 แนวคิดหลักที่นำปรับใช้ คือ
        1. สามารถปรับตัวเข้ากับสถาพแวดล้อมและการใช้งานที่เปลี่ยนแปลงได้ดี
        2. พัฒนาและปรับปรุงศิลปะแห่งการเขียนโปรแกรม

    โดยมันเกิดมาจากความต้องการที่จะแก้ไขปัญหาของภาษารุ่นก่อน ๆ (c, c++)

    ปัญหาของโปรแกรมสมัยก่อน มักจะออกแบบด้วยหลักการ GOTO เป็นวิธีหลัก ทำให้เกิดปัญหา โค้ดสปาเก็ตตี้

    แล้ว Java เกิดขึ้นมาเพื่อแก้ปัญหาอะไร
        ในสมัยก่อนที่จะเกิด Java เราสามารถใช้งาน c/c++ ได้ทุกอุปกรณ์ แต่การจะคอมไพลชุดคำสั่งให้สำหรับแต่ละ cpu
        นั้นมีราคาที่ค่อนข้างแพง เพราะเราจะต้องนำ cpu ของแต่ละสถาปตยกรรมมาคอมไพลเอง จึงเกิดแรสคิดที่จะสร้าง
        ภาษาที่คอมไพลรอบเดียว สามารถใช้งานได้ทุกที่ ภายใต้สภาพแวดล้อมที่แตกต่างกัน

        โดย java สามารถให้เราควบคุมชุดคำสั่งได้เต็มที่ ทำให่้โปรแกรมจะออกมาดีไม่ดี อยู่ที่โปรแกรมเมอร์จะเขียนชุดคำสั่งขึ้นมา
