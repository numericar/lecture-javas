Java Bytecode
    กุญแจสำคัญที่ทำให้ java สามารถแก้ไขปัญหาด้านความปลอดภัยและความสามารถในการพกพาได้คือคอมไพลเลอร์ของ java
    โดย java จะถูกคอมไพลออกไปเป็นเป็น bytecode ซึ่งคือชุดคำสั่งที่ถูกออกแบบขึ้นมาให้เพื่อระบบ runtime ที่ชื่อว่า jvm
    (java virtual machine) ทำงานได้ โดยอุปกรณ์อะไรก็ตามที่มี jvm ติดตั้งอยู่ จะสามารถทำให้ bytecode สามารถทำงานได้
    บนอุปกรณ์นั้น ๆ 

    และ jvm ยังมีข้อดีอีกคือจะคอยป้องกันไม่ให้ชุดคำสั่งที่ได้รับจาก bytecode สร้างผลข้างเคียงให้กับโปรแกรมอื่น ๆ ที่อยู่นอกระบบ
    ได้รับผลกระทบ

    แต่จะมีข้อเสียเล็กย้อบ คือ โปรแกรมที่ทำงานด้วย virtual machine อาจทำงานช้ากว่าโปรแกรมที่ถูกคอมไพลให้กับ cpu นั้นโดยตรง

    " เขียนเพียงครั้งเดียว รันได้ทุกที่ ทุกเวล ตลอดไป "
