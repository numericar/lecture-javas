ในการจะเขียน java program เราจำเป็นจะต้องจัดเตรียมเครื่องมือและทรัพยากรณ์ที่เหมาะสม
โดยเครื่องมือที่เราจะต้องจัดเตรียม คือ
    1. JDK (Java Development Kit)
            โดยมันจะเตรียมคอมไพลเลอร์และรันไทม์ไว้ให้ ซึ่งทำให้เราสามารถดำเนินการกับ java program ได้ 
        และสามารถ debug program ได้ด้วย

    2. JRE (Java Runtime Environment)
            เป็นส่วนหนึ่งของ JDK รวยรวม class library พื้นฐานของ java 

    3. JVM (Java Virtual Machine)
            เป็นส่วนหนึ่งของ JRE เป็นสิ่งที่คอยดำเนินการคำสั่ง java ของเรา
