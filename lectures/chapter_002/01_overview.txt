Overview of Java

    Object-Oriented Programming
        เป็นการเขียนโปรแกรมเชิงวัตถุ ซึ่งเป็นหัวใจสำคัญของภาษา java โดยโปรแกรมทั้งหมดที่เขียนด้วยภาษา java ล้วนเป็น oop ทั้งหมด

    Two Paradigms
        โปรแกรมคอมพิวเตอร์ทั้งหมดบนโลกใบนี้ ประกอบด้วยาส่วนประกอบเพียง 2 ส่วน คือ 
            1. ชุดคำสั่ง
            2. ข้อมูล
        
    โดย Java ใช้ความสามารถของ oop ในการจัดการกับความซับซ้อนในการเขียนโปรแกรม โดยเรามักจะใช้มันจะระเบียบโปรแกรม
    เช่น การกำหนดชุดของ interface เพื่อบอกว่า เราสามารถทำอะไรกับข้อมูลนั้นได้บ้าง

    Abstraction
        นามธรรม เป็นแนวคิดที่ใช้ซ่อนรายละเอียดที่ซับซ้อน ตัวอย่างเช่น เมื่อเรามองไปที่รถยนต์ เราไม่ได้คิดว่า มันเป็นวัตถุชิ้นหนึ่งที่มีวัตถุอื่น ๆ
        มากมายเป็นหมื่นชิ้นมาประกอบรวมกัน เราคิดค่าว่า รถคันนี้เคลื่อนที่ยังไง ความเร็วเท่าไหร่ เบรกยังไง นั่งได้กี่คน
        โดยผู้ใช้สามารถขับรถไปยังจุดหมายได้ โดยไม่ต้องรู้สึกสับสนกับชิ้นส่วนนับหมื่นชิ้นของรถ แต่ผู้ใช้ยังสามารถใช้งานชิ้นส่วนของรถได้แบบอิสระ

    หลักการ oop 3 ประการ
        การเขียนโปรแกรมเชิงวัตถุมีกลไกอยู่ 3 ประการ คือ 
            1. การห่อหุ้ม (encapsulation)
            2. การสืบทอด (inheritance)
            3. polymorphism (ความหลากหลาย)

    Encapsulation
        เป็นการรวมชุดคำสั่ง และ ข้อมูลที่ชุดคำสั่งนั้นจะต้องจัดการเข้าด้วยกัน และจะทำการปกป้องข้อมูลเหล่านั้น ไม่ให้สามารถถูกจัดการได้จากภายนอก
        เพื่อป้องกันการใช้งานที่ไม่เหมาะสม โดยหากจะทำการเข้าถึงข้อมูล จะต้องเข้าถึงผ่านฟังก์ชันที่จัดเตรียมไว้ให้เท่านั้น

        ตัวอย่างในโลกของความเป็นจริง เมื่อเราใช้รถระบบเกียร์อัตโนมัติของรถยนต์ ระบบจะจัดการกับข้อมูลจำนวนมากที่เกี่ยวกับเครื่องยนต์ เช่น 
        ระดับการเร่ง ความละเอียดของพิ้นถนน ตำแหข่งของคันเกียร์ โดนเราในฐานะผู้ใช้สามารถควบคุมข้อมูลที่ซับซ้อนนี้ได้ด้วยวิธีเดียว คือ การเลื่อนคัยเกียร์

    ** ข้อมูลที่คลาสกำลังจัดเก็บ เรียกว่า ตัวแปร

    Inheritance
        เป็นการที่วัตถุหนึ่งได้รับคุณสมบัติจากวัตถุอีกตัวหนึ่ง ซึ่งมีความสำคัญมาก เพราะมันสนับสนุนแนวคิดการจัดหมวดหมู่แบบลำดับชั้น (hierarchical classification) จากบนลงล่าง
        ตัวอย่างเช่น สุนัขพันธุ์ไซบีเรียน เป็นส่วนหนึ่งของหมวดหมู่ สุนัข ซึ่งอยู่ภายใต้หมวดหมู่ สัตว์เลี้ยงลูกด้วยนม และ สัตว์เลี้ยงลูกด้วยนม ก็อยู่ภายใต้หมวดหมู่ที่ชื่อว่า สัตว์
